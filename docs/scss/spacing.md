# Spacing


## Padding

```scss
@include padding(10,20,30,40,true);          // padding: 10rem 20rem 30rem 40rem !important;
@include padding(10px,20px,30px,40px,true);  // padding: 10px 20px 30px 40px !important;

@include padding-top(3px);                   // padding-top: 3px;
@include padding-right(3px);                 // padding-right: 3px;
@include padding-bottom(3px);                // padding-bottom: 3px;
@include padding-left(3px);                  // padding-left: 3px;
@include padding-x(3px);                     // padding-right: 3px; padding-left: 3px;
@include padding-y(3px);                     // padding-top: 3px; padding-bottom: 3px;
```
